Unter der Organisation von Jörg Reuter, Studienrat der Ferdinand-Braun-Schule und der kompeteten Moderation von Ralf Kehl konnten Interessierte, am Freitag, 23. Juni 2017 an einem spannenden und informativen Workshop über REACT teilnehmen.

REACT ist eine Bibliothek, die es ermöglicht, Javascript auch auf dem Server einzusetzen. Sie ist auf gutem Weg PHP im Backend abzulösen.

JavaScript ist als weit verbreitete Internet-Programmiersprache in nahezu jeder Webanwendung zu finden. Mit JavaScript programmierte Anwendungen können mit wenig Aufwand zu einer App für Android oder iOS umgewandelt werden.

Die Chancen auf dem Arbeitsmarkt für den gut bezahlten Beruf des Anwendungsentwicklers im Bereich JavaScript sind überdurchschnittlich hoch.

Danke an den Verein Zeitsprung e.V. für die Bereitstellung der Räumlichkeiten :-).

Wir, das Fachinformatiker-Forum Fulda, treffen uns in regelmäßigen Abständen im Technologiezentrum ITZ.

Nach den Sommerferien freuen wir uns auf einen spannenden Workshop zum Thema „IT Security“ und hoffen, Euch dort begrüßen zu können. Bis dahin eine gute Zeit!

Die Termine könnt Ihr auf unserer Website http://fachinformatiker-forum.de einsehen .
