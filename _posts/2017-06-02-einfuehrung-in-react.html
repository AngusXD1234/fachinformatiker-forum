React ist ein modernes JavaScript Framework um HTML Web-Applikationen zu erstellen. Die Erweiterung 'React Native' ermöglicht zudem die Entwicklung von nativen Applikationen für Android und iOS.
Die Veranstaltung vermittelt einen Querschnitt und ersten praktischen Einstieg in die Konzepte der JavaScript React Welt.
Rendern mit Componenten Lifecycle, JSX, States und Props.

Vorgestellt werden u.a. grundlegende Programmiertechniken von Router, Redux und ReactNative.
